## 🎯 Plan6核心愿景

**从Agent工具进化到AI投资合伙人**

> **"每个价值投资者都值得拥有一个AI投资团队"**

### 核心理念

Plan6不是一个交易平台或工具系统，而是一个**AI投资合伙公司**：

- **AI作为合伙人** - Claude是你的智能投资合伙人，不是工具
- **Subagent作为团队** - 专业AI团队协作决策（研究、分析、交易、风控）
- **MCP作为连接** - 统一连接所有数据源和交易接口
- **巴菲特模式** - 1956-1969合伙模式的AI化实现
- **价值投资导向** - Graham-Buffett-Munger三位一体深度整合

## 💎 6大核心创新

### 1. AI投资合伙人概念 ⭐ (业界首创)

**巴菲特合伙模式AI化** (1956-1969):

```text
巴菲特合伙模式:
- 合伙人出资
- 巴菲特管理投资
- 超过6%收益部分分成25%

Plan6 AI化:
- 投资者出资
- AI团队管理投资
- 超过基准收益AI分成20%

核心区别:
AI不是工具，而是真正的合伙人
共同决策，共担风险，共享收益
```

### 2. Subagent专业团队 ⭐ (官方最佳实践)

基于**Claude Agent SDK官方Subagent最佳实践** (2025):

```text
Research Team (研究团队) - 4个Subagents并行:
├─ FundamentalResearcher (基本面研究)
├─ TechnicalAnalyst (技术分析)
├─ SentimentAnalyst (情绪分析)
└─ MacroAnalyst (宏观分析)

Analysis Team (分析团队) - 4个Subagents分层:
├─ ValuationAnalyst (估值分析 - Planner)
├─ QualityAnalyst (质量分析)
├─ RiskAnalyst (风险分析)
└─ MoatAnalyst (护城河分析)

Trading Team (交易团队) - 3个Subagents执行:
├─ ExecutionAgent (执行Agent)
├─ PositionSizer (仓位管理 - Kelly)
└─ OrderRouter (订单路由)

Risk Team (风控团队) - 3个Subagents监控:
├─ PortfolioMonitor (组合监控)
├─ RiskManager (风险管理)
└─ ComplianceAgent (合规检查)

Chief Investment Agent (AI巴菲特) - Supervisor:
└─ 最终决策，团队协调
```

### 3. MCP统一架构 ⭐ (开放标准)

基于**Model Context Protocol开放标准** (2025):

```text
MCP Gateway (统一网关)
    │
    ├─ Data Sources (数据源MCP)

        └─ Slack (消息通知)

优势:
✓ 统一接口
✓ 热插拔
✓ 社区生态
✓ 开放标准
```

### 4. 科学仓位管理 ⭐ (三合一)

**Kelly准则 + MPT优化 + Munger集中投资**:

```text
Kelly准则 (最优增长):
f* = (预期收益 - 无风险利率) / 方差
使用半Kelly降低波动

MPT优化 (风险控制):
均值-方差优化
有效前沿计算
约束条件处理

Munger集中 (3-5个最佳机会):
第一名: 40%
第二名: 30%
第三名: 20%
第四名: 10%
第五名: 5%

三合一决策:
最终仓位 = Kelly × 30% + MPT × 30% + Munger × 40%
```

### 5. 价值投资三位一体 ⭐ (最完整)

**Graham-Buffett-Munger完整体系**:

```text
Graham (深度价值):
├─ Graham公式: V = EPS × (8.5 + 2g)
├─ Net-Net筛选
├─ 安全边际: 30-40%
└─ 深度折价买入

Buffett (质量价值):
├─ ROIC > 10%
├─ 宽护城河
├─ 优秀管理层
└─ 公允价格 (不必深度折价)

Munger (多元思维):
├─ Lollapalooza效应检测
├─ Mental Models应用
├─ 能力圈原则
└─ 逆向思维

三位一体共振:
✓ Graham安全边际30%+
✓ Buffett ROIC>10% + 护城河
✓ Munger Lollapalooza效应
= 重仓买入机会 (30-50%仓位)
```

### 6. 混合编排模式 ⭐ (灵活高效)

基于**2025年Multi-Agent最佳实践**:

```text
Supervisor模式:
AI巴菲特总协调，分发任务，收集结果，最终决策

Hierarchical模式:
Planner制定计划 → Executors执行 → 聚合结果

Parallel模式:
Research Team并行工作，提高效率

Debate模式:
Trading Team (支持买入) vs Risk Team (风险提示)
→ Supervisor作为裁判裁决

Hybrid Orchestrator:
根据任务类型动态选择最佳编排模式
```
